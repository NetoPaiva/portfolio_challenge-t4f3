### Detalhes do desafio

1. Conteúdos importantes (links abaixo)

  - Modelo Figma
    - design das telas/dispositivos
  - Trello - Sprint 01 - Portfolio
    - instruções / roteiro do projeto
  - Discord
    - comunidade dev ONE
  - 4. Alura+
    - como front-end utiliza o Figma
    - vídeo / instruções


2. Cards do Trello

  - Ferramenta de gestão de desenvolvimento do projeto
  - Seguir o quadro original Challenge ONE / Alura
  - copiar o quadro para gerenciar andamento do projeto


3. Sugestão Implementação / Cursos / Formações

  - HTML / CSS / HTML

    - Flexbox
    - Responsividade
    - Arquitetura CSS
    - Acessibilidade
    - Validação formulário


4. Início do projeto

  - Baixar projeto base  no repositório do GitHub Alura
    - index.html [finalizado]
    - style.css [instruções/sugestões de estilo]
    - validacao.js [em branco]
      - aplicar lógica de programação
      - validação do formulário


5. Entrega do projeto

  - Publicar repositório / página no GitHub
    - como publicar com GitHub Pages

  - Adicionar repositório no tópico "#challengeoneportfolio4"
    - perfil do GitHub
    - menu lateral direito do repositório
    - aba Sobre / About
    - na engrenagem
    - preencher o campo "Tópico / Topics"
      "challengeoneportfolio4"
    - salvar alterações

  - Entrega final
    - preencher formulário com dados / link
      - repositório GitHub
      - publicação GitHub Pages
    - receber badge da conquista por email
    - compartilhar projeto
      "#Oracle #AluraLatam"
