## Analisar layout


### Estrutura básica dos Layouts

  - Grid (colunas)
    - 1 coluna = 80px
    - entre colunas (gutter) = 16px

  - Mobile
    - 4 colunas (w: 360px)
    - margens: 16px

  - Tablete
    - 8 colunas (w: 768px)
    - margem esquerda/direita: 32px

  - Desktop
    - 12 colunas (w: 1440px)
    - margem esquerda/direita: auto


## Desmembrar elementos

  - capturar telas dos elementos (OK)


### Header elementos

  - versão mobile
    - nome (logo)
    - menu navegação
      - email (normal)

  - versão tablete
    - nome (logo)
    - menu navegação
      - email (bold)

  - versão desktop
    - nome (logo)
    - menu navegação
      - sobre mim
      - skills
      - hobbies
      - formação
      - projetos
      - email (bold)


### Banner (apresentação)

  - versão mobile
    - alinhado à esquerda
      - imagem de perfil (1,5 coluna)
    - alinhado à esquerda (largura total = 4 colunas)
      - título de apresentação
      - texto com breve descrição pessoal
        - nome
        - idade
        - nacionalidade
        - biografia (+/- 50 caracteres)
      - linha com redes sociais

  - versão tablete
    - alinhado à esquerda (6 colunas)
      - título de apresentação
      - texto com breve descrição pessoal
        - nome
        - idade
        - nacionalidade
        - biografia (+/- 50 caracteres)
      - linha com redes sociais
    - alinhada à direita
      - imagem de perfil (2 colunas)

  - versão desktop
    - alinhado à esquerda (8 colunas)
      - título de apresentação
      - texto com breve descrição pessoal
        - nome
        - idade
        - nacionalidade
        - biografia (+/- 50 caracteres)
      - linha com redes sociais
    - alinhada à direita
      - imagem de perfil (4 colunas)
